# Тренажёр: сложение и вычитание

Онлайн-тренажёр по математике (сложение и вычитание) с тремя уровнями и тремя режимами. Результаты можно отправлять в Google Таблицу через форму.

## Как разместить в интернете (GitHub Pages)

1. Создайте репозиторий на GitHub (например `math-trainer`).
2. Загрузите в него файл `index.html` (в корень репозитория или в папку `docs`).
3. В настройках репозитория: **Settings** → **Pages** → в разделе **Source** выберите ветку (например `main`) и папку (корень или `/docs`).
4. Сохраните. Через минуту сайт будет доступен по адресу вида `https://ВАШ_ЛОГИН.github.io/ИМЯ_РЕПОЗИТОРИЯ/` (если файл в корне — то `.../ИМЯ_РЕПОЗИТОРИЯ/`, если в папке `docs` — укажите в Source папку `docs`).
5. Эту ссылку можно отправлять ученикам.

**Альтернатива:** зайдите на [netlify.com/drop](https://app.netlify.com/drop), перетащите папку с `index.html` — получите готовую ссылку на сайт.

## Настройка Google Form для приёма отчётов

1. Создайте новую форму в [Google Формах](https://forms.google.com).
2. Добавьте вопросы (поля):
   - **Имя** (короткий ответ)
   - **Режим** (короткий ответ)
   - **Уровень** (короткий ответ)
   - **Правильных** (короткий ответ)
   - **Всего** (короткий ответ)
   - **Время** (короткий ответ)
   - **Дата** (короткий ответ)
3. Включите сохранение ответов в таблицу: **Ответы** → **Связать с Таблицей** → создать новую таблицу или выбрать существующую.
4. Получите ссылку предзаполнения: нажмите **Отправить** → иконка **</>** (встраивание) или **⋮** → **Предзаполнить форму**. Откроется ссылка с параметрами вида `entry.123456789=значение`.
5. Откройте `index.html` в редакторе и в начале блока `<script>` найдите константы:
   - **GOOGLE_FORM_BASE_URL** — вставьте сюда ссылку на форму до символа `?` (например `https://docs.google.com/forms/d/e/1FAIpQLS.../viewform`). Если в ссылке уже есть `?usp=pp_url`, оставьте как есть.
   - **GOOGLE_FORM_ENTRIES** — замените каждый `entry.XXXXXX` на реальный ID поля из ссылки предзаполнения (для каждого вопроса в форме будет свой `entry.XXXXXXXXX`).
6. Сохраните `index.html` и снова загрузите его на GitHub (или Netlify), если сайт уже был опубликован.

После этого кнопка «Отправить результат учителю» будет открывать форму с уже подставленными данными; ученику останется нажать «Отправить», и ответ появится в вашей Google Таблице.
